# The Quantum Wall

A Vite + React + TypeScript project with shadcn/ui components.

## Setup

```bash
npm install
npm run dev
```

## Pages

- `/` — Landing page with "The Quantum Wall" hero and **Test Now** button
- `/test` — API Playground with two side-by-side forms:
  - **Left**: Qubit Measurement form
  - **Right**: Entanglement Query form
  - Each shows its response below after submission

## Tech Stack

- Vite 5 + React 18 + TypeScript
- Tailwind CSS
- shadcn/ui (Button, Input, Label, Textarea, Card)
- React Router v6
- Syne + DM Sans + JetBrains Mono fonts
